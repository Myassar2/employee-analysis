# Employee Analysis Project 📊

## 📌 Overview
This project analyzes employee data to extract insights about workforce distribution, employee turnover, and demographics.

## 💡 Objectives
- Analyze employee distribution by department
- Study employee exit patterns
- Explore gender and age distributions
- Identify key trends in the company

## ⚙️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📂 Dataset
The dataset includes:
- Employee demographics (Age, Gender)
- Department information
- Hire and exit dates
- Employment status

## 📊 Key Analysis Performed
- Age group categorization
- Department distribution
- Employee turnover analysis
- Gender distribution of employees who left

## 📈 Key Insights
- Some departments have higher employee turnover
- Gender distribution varies among exited employees
- Age groups show different patterns in employment

## ▶️ How to Run
1. Open `Employee.ipynb`
2. Run all cells
3. View visualizations and results

## 📌 Notes
This project is part of a data analysis practice to improve data cleaning, visualization, and insight extraction skills.
